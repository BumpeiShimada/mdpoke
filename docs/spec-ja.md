# mdpoke 仕様メモ

## チェックボックス

- タスクリストは `- [ ]`、`* [ ]`、`+ [ ]` のいずれの bullet marker でもチェックボックスとして描画する。
- `- [x]`、`* [x]`、`+ [x]` はチェック済みチェックボックスとして描画する。
- `Tab` / `Shift+Tab` のチェックボックス移動対象には、`-`、`*`、`+` で書かれたタスクリストをすべて含める。
- フォーカス中のチェックボックスは `Space` または `Enter` で切り替えられる。
- チェックボックスを切り替えても、元の raw 行で使われていた `-`、`*`、`+` の bullet marker は保持する。
- マウスでチェックボックス自体をクリックした場合も、その行だけを切り替える。

## リンク表示とコピー

- Markdown に URL がそのまま書かれている場合は、URL 文字列を表示し、URL 用の色と下線で描画する。
- `<https://...>` のようなオートリンクも、URL 文字列を表示対象として扱う。
- `[リンクテキスト](https://...)` または `[リンクテキスト](http://...)` のような外部リンクは、URL を横に表示しない。
- 外部リンクの表示対象は `[]` 内のリンクテキストだけにし、裸 URL と同じ色と下線で描画する。
- 外部リンクの表示テキストをクリックした場合は、非表示の URL をクリップボードへコピーする。
- 外部リンクにフォーカスして `y` を押した場合も、非表示の URL をコピーする。
- ドラッグ選択で外部リンク行をコピーする場合は、表示上は隠れている URL をコピー文字列へ補完する。
- `#見出し` への内部アンカーリンクは外部リンクと区別できる見た目のままにし、クリック時はジャンプ確認を出す。

## 崩してはいけない挙動

- 裸 URL の表示、折り返し、クリックコピーは従来通り維持する。
- コードスパンやコードブロック内の `[text](https://...)` は Markdown リンクとして扱わず、書かれた文字列をそのまま表示する。
- 内部アンカーリンクと外部リンクのクリック動作は分ける。内部アンカーはジャンプ、外部リンクはコピーを行う。
